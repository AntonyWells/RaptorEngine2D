RaptorEngine2D by AntonyWells

A very advanced open source tool kit to facilitate the creation of next generation 2d games and tools.

RaptorEngine2D is wrote primarly in and for the BlitzMax programming language. It features an advanced rotating/scaling lit tile based map engine, using tile sheets. It can load Tiled xml based maps(the new qt version) natively. Features a nice auto-packing content system.

Dependencies

BlitzMax,OpenGL,GLSL,Shader Model 3

Install

No installation presently, simply run the demo project called culture. a Blide solution is included.
